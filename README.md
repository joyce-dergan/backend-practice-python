# File Integrity Checker

Projeto desenvolvido como exercício prático utilizando Python.

## Objetivo
Criar um sistema capaz de verificar a integridade de arquivos por meio da comparação de hashes.

## Tecnologias utilizadas
- Python

## Aprendizados
- Manipulação de dados
- Lógica de programação
- Comparação de hashes
- Boas práticas de código

## Como executar
1. Clone o repositório  
2. Execute o arquivo principal com Python
